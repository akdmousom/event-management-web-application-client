###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript

const nums = [0, 1, 2, 3];

nums.forEach((num) => {
  if (num) count += 1;
});

console.log(count);
```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C

<i>A variable is taken here that is count and its value is 0. We know that 0 is equivalent to false in javascript. Another array is taken. When the foreach loop is executed on that array, one element at a time is returned to num. And the if condition given here is checking how many true elements there are</i>

</p>
</details>
